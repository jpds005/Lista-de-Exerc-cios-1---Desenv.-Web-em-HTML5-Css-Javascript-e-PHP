Qual é a diferença entre linguagem de programação Server Side e Client Side?

Client Side X Server Side

  • CLIENT SIDE: se refere a tecnologias e códigos disponibilizados no lado do cliente, ou seja, operações que são executadas no navegador do usuário para gerar a página que ele interage na web
  • SERVER SIDE: se refere a tecnologias e códigos disponibilizados no lado do servidor, ou seja, operações que responderão as requisições do cliente, processando as solicitações e enviando de volta os conteúdos requisitados

Exemplos práticos de cada tipo:

  • NO CLIENT SIDE: Um exemplo fundamental de tecnologia do client side é a estruturação de páginas web com a linguagem HTML. O acabamento pode ser feito com a liguagem CSS. A interatividade pode ser proporcionada numa integração com a linguagem JavaScript
  • NO SERVER SIDE: Um exemplo fundamental de tecnologia do server side é permitir o acesso a informações armazenadas em bancos de dados. Uma vez que apenas utilizando HTML e JavaScript isso não é possível, faz-se necessária a utilização de uma linguagem no lado servidor, como PHP e Node.js
